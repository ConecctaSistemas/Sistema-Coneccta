Leitura de etiqueta de balança no PDV

Configure como o caixa PDV interpreta os códigos EAN-13 impressos pela balança. Toledo, Urano e Filizola usam prefixo "2" para produtos pesados.

Estrutura do código de barras Preencha os parâmetros acima para visualizar o formato.

Como funciona: Cadastre o PLU no produto (aba Outras → Balança) · Gere o arquivo "Somente produtos de balança" e envie à balança · No caixa, ao passar a pistola o peso/preço é lançado automaticamente. Suportadas: Toledo Prix-3/4 · Urano Prisma/POP/URF32 · Filizola MK/Prix · qualquer modelo EAN-13 com prefixo interno.
